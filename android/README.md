# Android APK (TWA wrapper)

This project is a PWA. On Android, users can install it from the browser (Add to Home Screen).

If you want a real APK, wrap the PWA as a **Trusted Web Activity** (TWA).

Recommended tool: **Bubblewrap** (by Google).

High-level steps:

1) Install Java + Android SDK (Android Studio).
2) Install Bubblewrap:
   - `npm i -g @bubblewrap/cli`
3) Build and deploy your website (GitHub Pages URL).
4) Initialize TWA:
   - `bubblewrap init --manifest=https://<your-domain>/manifest.webmanifest`
5) Build APK:
   - `bubblewrap build`

Notes:
- Use the deployed HTTPS URL (TWA requires HTTPS).
- The app icon will come from `manifest.webmanifest` (generated by VitePWA).
